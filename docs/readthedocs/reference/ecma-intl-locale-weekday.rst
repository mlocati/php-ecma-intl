===========================
Ecma\\Intl\\Locale\\WeekDay
===========================

.. php:namespace:: Ecma\Intl\Locale
.. php:enum:: WeekDay : int

    Days of the week.

    Each case is mapped to an integer value corresponding to the values defined in
    `WeekInfo Record Fields <https://tc39.es/proposal-intl-locale-info/#table-locale-weekinfo-record>`_.

    .. php:case:: Monday : 1
    .. php:case:: Tuesday : 2
    .. php:case:: Wednesday : 3
    .. php:case:: Thursday : 4
    .. php:case:: Friday : 5
    .. php:case:: Saturday : 6
    .. php:case:: Sunday : 7
